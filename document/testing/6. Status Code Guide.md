### 2. Use Environment Variables

Buat environment untuk dev dan production:

**Development**:
```

```

**Production**:
```
base_url: https://api.tokomainan.com/api
```

Switch environment dengan dropdown di kanan atas.

### 3. Use Pre-request Scripts

Auto update token di Collection â†’ Tab **Scripts**:

```javascript
// Pre-request Script (Collection level)
const token = pm.environment.get("access_token");
if (token) {
    pm.request.headers.add({
        key: "Authorization",
        value: "Bearer " + token
    });
}
```

### 4. Use Tests Tab

Auto save token setelah login:

```javascript
// Tests (pada request Login)
if (pm.response.code === 200) {
    const response = pm.response.json();
    pm.environment.set("access_token", response.access_token);
    console.log("Token saved!");
}
```
