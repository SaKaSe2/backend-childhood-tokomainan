### Mengapa Inherit Auth?

Daripada copy-paste token ke setiap request, lebih baik setup di Collection level.

### Setup Steps:

#### 1. Setup di Collection Level

1. Klik Collection `Toko Mainan API`
2. Tab **Authorization**
3. Type: `Bearer Token`
4. Token: `<kosong dulu>` (akan diisi setelah login)
5. **Save**

#### 2. Setup di Request Level

Untuk setiap request yang butuh auth:

1. Tab **Authorization**
2. Type: `Inherit auth from parent`
3. **Save**

### Cara Update Token:

Setelah login dan dapat token:
1. Copy `access_token` dari response
2. Klik Collection `Toko Mainan API`
3. Tab **Authorization**
4. Paste token di field **Token**
5. **Save**

Semua request yang `Inherit auth from parent` otomatis pakai token ini! ğŸ‰
