### ‚ùå Error: "Database connection refused"

**Possible Causes**:
1. MySQL service tidak berjalan
2. Database belum dibuat
3. Credentials salah

**Solutions**:

1. **Check MySQL Service**
```bash
# Windows (XAMPP)
# Start MySQL di XAMPP Control Panel

# Linux
sudo service mysql start

# Mac
brew services start mysql
```

2. **Create Database**
```sql
CREATE DATABASE backend_tokomainan;
```

3. **Verify .env**
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=backend_tokomainan
DB_USERNAME=root
DB_PASSWORD=
```

4. **Test Connection**
```bash
php artisan migrate
```
