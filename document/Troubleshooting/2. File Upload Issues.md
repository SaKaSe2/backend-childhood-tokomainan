### âŒ Error: "The file failed to upload"

**Possible Causes**:
1. Symbolic link belum dibuat
2. File size melebihi limit
3. Permission error

**Solutions**:

1. **Create Symbolic Link**
```bash
php artisan storage:link
```

Verify:
```bash
ls -la public/storage
# Harus ada symlink ke storage/app/public
```

2. **Check File Size**
   - Limit: 5MB (5120 KB)
   - Cek file size sebelum upload
   - Edit validation jika perlu lebih besar

3. **Fix Permissions**
```bash
# Linux/Mac
chmod -R 775 storage/
chmod -R 775 bootstrap/cache/

# Atau
sudo chown -R www-data:www-data storage/
sudo chown -R www-data:www-data bootstrap/cache/
```

### âŒ Error: "File URL returns 404"

**Cause**: File tidak dapat diakses via URL

**Solutions**:

1. **Verify Symlink**
```bash
php artisan storage:link
```

2. **Check File Path**
```php
// Response harus seperti ini
"file_path": "products/1702809600_image.jpg"

// URL akses
http://localhost:8000/storage/products/1702809600_image.jpg
```

3. **Check File Exists**
```bash
ls storage/app/public/products/
# File harus ada di sini
```

---

### âŒ Error: "Maximum upload size exceeded"

**Cause**: File terlalu besar (>5MB)

**Solutions**:

1. **Compress File** (recommended)
   - Gunakan tools seperti TinyPNG
   - Resize image jika terlalu besar

2. **Increase Limit di Laravel**
Edit validation:
```php
'file' => 'nullable|file|max:10240' // 10MB
```

3. **Increase PHP Limits**
Edit `php.ini`:
```ini
upload_max_filesize = 10M
post_max_size = 10M
```

Restart server:
```bash
# XAMPP: Stop & Start Apache
# Artisan: Restart php artisan serve
```
