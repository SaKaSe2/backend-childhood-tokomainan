### ‚ùå Error: "JWT_SECRET not set"

### ‚ùå Error: "Token has expired"

**Cause**: Token JWT sudah melewati expire time (default 60 menit)

**Solution**:
1. Login ulang untuk mendapat token baru
2. Update token di Postman Collection Authorization

**Cara Extend Expiration**:
Edit `config/jwt.php`:
```php
'ttl' => 1440, // 1 hari (dalam menit)
```

---

### ‚ùå Error: "Unauthenticated" atau "Token invalid"

**Possible Causes**:
1. Token tidak disertakan di header
2. Format token salah
3. Token sudah expired
4. Token corrupted

**Solutions**:

1. **Cek Header Format**
```
Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGc...
```
- Harus ada kata "Bearer " (dengan spasi)
- Token tidak boleh ada quotes

2. **Cek di Postman**
   - Tab Authorization ‚Üí Type: Bearer Token
   - Paste token tanpa "Bearer " prefix

3. **Login Ulang**
```bash
# Test login via Postman
POST /api/auth/login
```

4. **Debug Token**
   - Decode di https://jwt.io/
   - Cek expiration time
   - Cek payload valid

---

### ‚ùå Error: "The token has been blacklisted"
