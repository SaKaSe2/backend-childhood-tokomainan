### 7Ô∏è‚É£ Products - Create (dengan File) üîí

```
Method: POST
URL: {{base_url}}/products
```

**Authorization**: `Inherit auth from parent`

**Body**: `form-data`

| Key | Type | Value |
|-----|------|-------|
| name | Text | LEGO City Police Station |
| description | Text | Set LEGO kantor polisi lengkap |
| price | Text | 1250000 |
| stock | Text | 15 |
| category | Text | lego |
| age_range | Text | 8-12 |
| brand | Text | LEGO |
| rating | Text | 4.5 |
| is_featured | Text | true |
| image_url | Text | https://example.com/image.jpg |
| file | File | [Select File] |

**Steps untuk Upload File**:
1. Klik di kolom Value untuk row `file`
2. Klik **Select Files**
3. Pilih **Select file from local machine**
4. Browse dan pilih file (max 5MB)
5. Klik **Send**

**Expected Response (201)**:
```json
{
  "status": "success",
  "message": "Product created successfully",
  "data": {
    "id": 4,
    "name": "LEGO City Police Station",
    "file_path": "products/1702809600_lego.jpg",
    ...
  }
}
```
