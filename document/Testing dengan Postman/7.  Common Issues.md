### Issue 1: Token Not Working

**Error**: `Unauthenticated` atau `Token invalid`

**Solutions**:
1. Pastikan format: `Bearer <token>`
2. Pastikan tidak ada space extra
3. Pastikan token belum expired (1 jam)
4. Login ulang untuk token baru

### Issue 2: File Upload Failed

**Error**: Validation error atau file not uploaded

**Solutions**:
1. Pastikan Body type: `form-data`
2. Pastikan file key type: `File` (bukan Text)
3. Cek file size < 5MB
4. Cek symbolic link: `php artisan storage:link`

### Issue 3: 404 Not Found

**Solutions**:
1. Cek URL endpoint benar
2. Cek server berjalan: `php artisan serve`
3. Cek routes: `php artisan route:list`

### Issue 4: CORS Error

Jika frontend React/Vue:

**Solution**: Install CORS package
```bash
composer require fruitcake/laravel-cors
```
