### 1. Create New Collection

1. Buka Postman
2. Klik **Collections** (sidebar kiri)
3. Klik tombol **+** atau **New Collection**
4. Nama: `Toko Mainan API`
5. Klik **Create**

### 2. Setup Base URL (Optional)

1. Klik Collection `Toko Mainan API`
2. Tab **Variables**
3. Add variable:
   - Variable: `base_url`
   - Initial Value: `http://localhost:8000/api`
   - Current Value: `http://localhost:8000/api`
4. **Save**

Sekarang bisa pakai: `{{base_url}}/products`
